#❤️ Heart-disease-prediction
 A machine learning model to predict the likelihood of heart disease using Logistic Regression on structured dataset.

FEATURES-
I) Exploratory Data Analysis (EDA) to understand dataset patterns
II) Data preprocessing (handling missing values, normalization, etc.)
III) Feature selection to identify key health indicators
IV) Logistic Regression model for classification
V) Stratified 80-20 train-test split for balanced evaluation
VI) Visualization using Matplotlib

TECH STACK-
-Python
-Scikit-learn – Machine learning model
-Pandas & NumPy – Data handling
-Matplotlib – Data visualization
